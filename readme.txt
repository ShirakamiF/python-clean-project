# Python Data Cleaning Project

## Overview
This project demonstrates how to clean and validate a messy sales dataset (200+ rows) using Python, Pandas, and NumPy.  
The goal was to standardize customer information, validate numeric fields, and recalculate totals for accurate analysis.

## Steps Taken
- Normalized customer names, emails, and phone numbers
- Validated ages, ratings, and quantities
- Standardized product categories and payment methods
- Recalculated `Total_Amount` with discounts applied
- Removed duplicates and handled missing values

## Tools Used
- Python 3
- Pandas
- NumPy
- OpenPyXL (for Excel export)

## Results
- Cleaned dataset saved as `cleaned_sales_data.xlsx`
- Ready for analysis and visualization in Power BI or Excel
